Написать приложение Paint 2.0

В окне приложения должен быть холст на поверхности которого будет происходить рисование и панель управления с инструментами.
Среди инструментов должны быть кнопки:
	- Линия
	- Прямоугольник
	- Эллипс
	- С заливкой(чек-бокс, для прямоугольника и эллипса)
	- Стереть
	- Очистить
	- Текст
	
Кроме того, должна быть возможность изменить толшину линии, цвет кисти и цвет фона.
Должна быть возможность загрузить существующее изображение вызвав OpenFileDialog нажатием на кнопку "Открыть"
И сохранить вызвав соответствующий диалог по кнопке "Сохранить"

При рисовании все нарисованное должно отображатся на холсте.
При рисовании фигур пользователь должен видеть как будет выглядеть итоговая фигура прежде чем отпустит кнопку мыши

При желании можно добавить другие инструменты рисования и кнопу "Отменить последнее действие"
