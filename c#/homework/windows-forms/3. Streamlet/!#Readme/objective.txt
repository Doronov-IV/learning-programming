Написать простой проводник аля-TotalComander.

Программа должна содержать два листбокса в котором должны отображаться папки и файлы.
По умолчанию при старте программы в каждом из листбоксов отображаются устройства.
У каждого листбокса должна быть "адресная строка". При введении правильного адреса 
с соответствующем листбоксе должна открытся введенная папка. Иначе - сообщение об ошибке.
Необходимо добавить ряд кнопок для создания, удаления, копирования и перемещения выбранных файлов или папок.
При выборе папки или файла в отдельных контролах для каждого листбокса должна отображаться 
вся доступная информация (размер, дата создания и т.д.) Кроме того, если выбранный файл является текстовым, 
то должна быть возможность открыть его и отобразить в отдельной форме.
Также, в отдельном окне должен открываться "Блокнот" с функцией ввода текста, сохранение
и загрузки текста через файловый диалог.
